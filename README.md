# ♞ Knight Tour: Andes

> Puzzle matemático basado en el Recorrido del Caballo de Ajedrez,  
> ambientado en las cumbres de los Andes ecuatorianos 🇪🇨

## Stack
- React 18 + TypeScript + Vite
- Ionic Capacitor 6 (Android / iOS)
- Firebase (Firestore + Auth)
- Google AdMob + RevenueCat (IAP)

## Features
- 🧠 Algoritmo Warnsdorff (solución garantizada)
- 🏔️ 15+ cumbres andinas con lore real
- ⚡ Power-Ups temáticos: Cóndor, Páramo, Cotopaxi
- 🔥 Racha diaria + Pasaporte Andino
- 📶 Modo campaña 100% offline
